Book Library Manager

URL: https://final-front-end-b2n9.onrender.com

This app allows you to:
- View all books in your library
- Add new books with details (title, author, genre, ISBN, rating, read status, notes)
- Edit existing book entries
- Delete books
- Search and filter your books
- View statistics about your collection
- Search for books using Open Library API and add them to your collection
